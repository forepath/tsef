Complete AI Agent Management Platform

Agenstra combines a powerful web-based IDE with distributed backend services to give you complete control over your AI agent infrastructure. Code directly in agent containers, access virtual workspaces with browser testing, manage agents, interact in real-time, and deploy infrastructure automatically - all from your browser.

Centralized Control

Connect to and manage multiple remote agent-manager services from a single console. The controller orchestrates all your distributed agent infrastructure in one place, enabling you to provision servers, monitor agent status across regions, and coordinate deployments - all from a unified dashboard.

Real-time Agent Interaction

WebSocket-based bidirectional communication with AI agents. Chat with agents, stream container logs, send commands in real-time through persistent connections, and control message flow with configurable filtering for content moderation and compliance.

Full-Featured Web IDE

Complete IDE experience powered by Monaco Editor (VS Code's editor). Syntax highlighting, IntelliSense, multi-file editing, Git integration, terminal access, live collaboration, and virtual workspaces with browser access - all in your browser.

Complete Agent Management Capabilities

Everything you need to create, manage, and interact with AI agents across distributed infrastructure.

Agent Lifecycle Management

Create, update, and delete AI agents through REST APIs. Each agent runs in its own isolated Docker container with secure authentication and credential management.

Real-time WebSocket Communication

Bidirectional WebSocket connections for chat messaging, container log streaming, and command execution. Maintain persistent connections with multiple agents simultaneously.

Full-Featured Web-Based IDE

Complete IDE experience with Monaco Editor (VS Code's editor). Syntax highlighting, IntelliSense autocomplete, multi-file tabs, Git diff viewer, markdown preview, terminal integration, and file tree navigation - all running in your browser.

Terminal & Container Management

Integrated terminal (xterm.js) for running commands, Docker-based agent execution with live log streaming, container stats monitoring, and full container lifecycle management - all within the IDE.

Integrated Git Workflow

Built-in Git manager with visual diff viewer, branch management, staging, commit, push, pull, rebase, and conflict resolution. All Git operations integrated directly into the IDE interface.

Automated Infrastructure

Provision cloud servers automatically with Docker and agent-manager deployment. Connect to Hetzner Cloud and other providers to spin up infrastructure in minutes.

Multi-Provider Agent Support

Plugin-based architecture supporting multiple AI agent providers. Use Cursor agents, OpenAI, Anthropic, or implement custom providers through a unified interface.

Centralized Dashboard

Single console interface to manage all your agent-manager instances. View agents across multiple servers, monitor status, and interact with any agent from one place.

Virtual Workspaces for Application Testing

Each agent can have its own separate virtual workspace container with a full desktop environment and Chromium browser to test web applications visually, debug browser interactions, and verify the agent's work - all accessible through a secure web-based VNC client.

  • Full Desktop Environment - Modern desktop with Chromium browser
  • Shared Workspace - Same files accessible from both IDE and browser
  • Web-Based Access - No client installation needed
  • Secure & Isolated - Password-protected VNC access with network isolation
